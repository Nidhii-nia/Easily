# Easily
A full-featured job portal application that connects job seekers with employers, featuring a clean UI, job search functionality, and application management.
Features
1. User Features
Job Browsing: Browse all available job listings

Advanced Search: Search jobs by keyword, location, skills, or company

Job Details: View comprehensive job information including salary, requirements, and company details

Easy Application: Apply for jobs with a simple form and resume upload

Application Confirmation: Receive email confirmation upon successful application

Responsive Design: Mobile-friendly interface for all devices

2. Employer/Recruiter Features
Job Posting: Post new job listings with detailed information

Job Management: Edit or delete existing job postings

Applicant Tracking: View all applicants for each job posting

Company Branding: Add company logo and details to job listings

Admin Dashboard: Manage job listings and applicants
